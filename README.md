# ⚡Tech Stack

**Front End :** ReactJS, Bootstrap

**API :** [OMDB API](https://www.omdbapi.com/)

## 🚀 Getting Started

Clone the project

```bash
  git clone https://github.com/darshangaikwad4114/Movie-app.git
```

Go to the project directory

```bash
  cd Movie-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
